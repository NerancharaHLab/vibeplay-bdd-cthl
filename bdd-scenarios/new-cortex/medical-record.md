## 1. เคสทดสอบที่ผ่านการเขียนสคริปต์อัตโนมัติแล้ว (Automated Scenarios)

### Scenario: Create a new patient with random ID
* **Given** Given the user is logged in to Cortex Cloud as a super user
* **When** When the user navigates to the Medical Record application
* **And** And the user starts creating a new patient record
* **And** And fills in the patient name, last name, and a generated ID card number
* **Then** Then the input fields should display the entered patient information correctly
